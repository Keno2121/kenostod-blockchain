## 1. Node Utility Model

Kenostod Validator Nodes perform four essential functions:

### 1.1 Course Completion Validation
- **Function:** Verify that students have legitimately completed course requirements
- **Process:** Nodes validate quiz scores, time spent, and assignment submissions
- **Reward:** Earn KENO for each validated completion
- **Impact:** Prevents fraudulent course completions and maintains platform integrity

### 1.2 KENO Reward Distribution
- **Function:** Process and distribute KENO token rewards to graduates
- **Process:** Nodes execute reward transactions on Binance Smart Chain
- **Reward:** Earn a percentage of distributed rewards (0.5% of each distribution)
- **Impact:** Enables scalable, decentralized reward processing

### 1.3 Flash Arbitrage Loan (FAL) Calculations
- **Function:** Run real-time arbitrage opportunity calculations across exchanges
- **Process:** Nodes fetch market data, calculate spreads, and identify opportunities
- **Reward:** Share of FAL platform fees (25% distributed to node operators)
- **Impact:** Powers the FAL/FALP systems that generate passive income for graduates

### 1.4 Graduate Credential Verification
- **Function:** Verify graduate status for employers, institutions, and third parties
- **Process:** Nodes respond to verification API requests with cryptographic proofs
- **Reward:** Earn KENO per verification request processed
- **Impact:** Creates trust layer for Kenostod credentials worldwide
