# KENO Token Logo Assets

Official emerald shield logo for the KENO cryptocurrency token.

## Available Formats

### Vector (Scalable)
- **keno-emerald-shield.svg** - Main logo (SVG format, infinite scaling)

### Raster Images (PNG)
Coming soon: Multiple sizes for different use cases
- 32x32px - Favicon, small icons
- 64x64px - UI elements, mobile
- 128x128px - Standard web display
- 256x256px - High-res web, marketing
- 512x512px - Print, large displays

## Usage

### Web/HTML
```html
<img src="/assets/keno-token/keno-emerald-shield.svg" alt="KENO Token" width="50" height="60">
```

### CSS Background
```css
.keno-logo {
  background-image: url('/assets/keno-token/keno-emerald-shield.svg');
  background-size: contain;
  width: 50px;
  height: 60px;
}
```

### Token Listings
When submitting KENO to exchanges or token platforms (CoinGecko, CoinMarketCap, etc.):
1. Download `keno-emerald-shield.svg`
2. Convert to required format (usually PNG 200x200 or 256x256)
3. Upload as token symbol/logo

## Export Instructions

### For External Platforms

**Download the SVG:**
1. Right-click on `keno-emerald-shield.svg` in Replit file explorer
2. Click "Download"
3. Save to your computer

**Convert to PNG (if needed):**
- Use online converters like CloudConvert, Convertio, or SVG2PNG
- Or use design tools like Figma, Adobe Illustrator, Inkscape
- Recommended sizes: 200x200, 256x256, 512x512

**For Binance Smart Chain (BEP-20):**
- Use 200x200 PNG format
- Upload to BscScan token info page

**For Exchange Listings:**
- Usually require 200x200 or 256x256 PNG
- Some accept SVG directly
