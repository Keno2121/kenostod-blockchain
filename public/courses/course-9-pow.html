⛏️ What is Mining?

Mining is the process of validating transactions and adding new blocks to the blockchain. Miners compete to solve complex mathematical puzzles, and the winner gets to add the next block and earn rewards.

Figure: The mining process — finding a valid hash through computational work

💡 Why Mining Matters

Mining serves two crucial purposes: (1) it secures the network by making attacks extremely expensive, and (2) it distributes new currency fairly to those who contribute computing power.

⚙️ How Proof-of-Work Works

The Mining Process

  • Gather Transactions: Collect pending transactions from the mempool
  • Create Block Header: Combine transactions, previous block hash, timestamp, and nonce
  • Hash the Block: Run SHA-256 on the block header to get a hash
  • Check Against Target: Is the hash below the difficulty target?
  • Increment Nonce: If hash doesn't meet target, try again with a new nonce
  • Broadcast Block: When valid hash found, broadcast to network and claim reward

Example Hash Puzzle

Target: Hash must start with "0000" (4 leading zeros) Attempt 1: nonce=0 Hash: 8f4a2c1e... ❌ (doesn't start with 0000) Attempt 2: nonce=1 Hash: 3b9e7d2a... ❌ (doesn't start with 0000) ...millions of attempts later... Attempt 4,892,371: nonce=4892371 Hash: 0000a7f3... ✅ VALID! Block mined!

📊 Difficulty Adjustment

The network automatically adjusts mining difficulty to maintain consistent block times, regardless of how much computing power joins or leaves the network.

  • Target Block Time: Kenostod aims for 1 block per minute
  • Too Fast? Increase difficulty (require more leading zeros)
  • Too Slow? Decrease difficulty (require fewer leading zeros)
  • Adjustment Period: Recalculated every 100 blocks

✅ Self-Regulating Network

This automatic adjustment means the network stays secure whether 10 or 10,000 miners are active, and blocks always arrive at a predictable rate.

💰 Mining Rewards

Miners earn rewards for successfully mining blocks:

  • Block Reward: Currently 100 KENO per block
  • Transaction Fees: All fees from included transactions
  • Halving Schedule: Block reward halves every 210,000 blocks

💡 Economic Incentives

Mining creates a positive feedback loop: miners invest in hardware → secure the network → earn rewards → fund more mining → network becomes more secure.
